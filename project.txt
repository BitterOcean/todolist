سلام دوستان.
توی اولین پروژه‌ای که قراره تحویل بدین، می‌خوایم یه to-do list طراحی کنیم به کمک react با مشخصات زیر:
۱. یه input / text field برای وارد کردن عنوان آیتم جدید.
۲. دکمه‌های add to bottom و add to top برای اضافه کردن یه آیتم به ابتدا یا انتهای لیست.
۳. لیست آیتم ها که هر کدومشون شامل یه چک‌باکس (برای تیک زدن اون آیتم و done کردنش)، عنوان آیتم و یه دکمه برای حذف اون آیتم رو دارن.
۴. آیتم‌های تیک خورده باید به یه شکل متفاوت نسبت به بقیه‌ی آیتم‌ها نشون داده بشن و به علاوه وقتی یه آیتم تیک می‌خوره باید بره ته لیست.

توی این پروژه اصل بر کدهای react هست که شما می‌زنین و UI اهمیت ویژه‌ای نداره. هر سوال یا ابهامی هم بود بپرسین.

در ضمن دوستانی که بدون دلیل این پروژه رو انجام ندن متاسفانه قادر به ادامه‌ی شرکت توی این دوره نیستن.
